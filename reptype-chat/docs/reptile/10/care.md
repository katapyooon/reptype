# ヘルマンリクガメ - 飼育方法

## ケージ

- アダルト1匹につき床面積90×45cm以上が理想。広いほどよい
- ベビー〜ヤングは45〜60cmのケージで飼育可能。成長に合わせて広げていく
- ベビーはケージが広すぎると餌場・水場を見失って弱ることがあるため注意
- 木製やガラス製のケースが扱いやすく保温性も確保しやすい
- ケージ内にはシェルターや水皿を置きつつ、リクガメが自由に歩けるルートを確保する

### 屋外飼育

お庭がある場合は囲いを作って屋外飼育も可能。日光浴がしやすくより自然に近い飼育ができる。

- 脱走防止のため囲いの高さを十分に確保し、地面を掘る習性に備えて柵の一部を地中に埋める
- 直射日光と日陰の両方を作ることが重要。流木や植木鉢でシェルターを設置し、カメが自分で体温調節できるようにする
- 外敵（カラス・猫）対策・脱走防止の蓋が必要

## 床材

砂や腐葉土など自然に近い素材を使用すると、掘る・歩くといった本来の行動を取りやすくなる。

- **砂＋腐葉土の混合**：適度な湿度を保ちやすく、自然に近い感触を再現できる
- **単体の砂**：乾燥系の管理に向いているが、湿度が下がりやすい

狭いケージや不適切な床材はストレスの原因になり、食欲不振や活動量の低下につながる。汚れた部分はこまめに取り除き、月1回を目安に全体を交換する。

## シェルター

ストレス軽減と体温調節のためにシェルターを設置する。

- ケージ内に隠れ家となるシェルターや流木を配置する
- バスキングスポット（暖かい場所）とクールスポット（涼しい場所）の両方に隠れ場所を確保すると理想的
- 屋外飼育では植木鉢や流木を利用した自然なシェルターも有効

## 照明

昼行性のため、日光を模倣するUVA/UVB照明が必要。

- **UVBライト**：紫外線を受けることでビタミンD3を生成し、骨・甲羅の形成に不可欠。必ず設置する。照射時間は1日12〜14時間が目安
- **バスキングライト**：輻射熱でケージ全体を暖める。光線が直線的すぎると甲羅の一部だけ乾燥が進み奇形につながるため、甲羅全体が暖まるものを選ぶ

UVBライトは見た目の明るさに関係なく紫外線量が徐々に低下するため、半年に1回を目安に交換する。天気の良い日は短時間の屋外日光浴も有効だが、熱中症を防ぐため日陰も確保すること。

## 温度管理

変温動物のため、ケージ内に「バスキングスポット（暖かい場所）」と「クールスポット（涼しい場所）」の温度勾配を作ることが重要。

| 時間帯・エリア | 温度目安 |
|--------|---------|
| バスキングスポット | 35〜40℃ |
| クールスポット | 24〜29℃ |
| 夜間 | 18℃以上を維持 |

- 夜間の保温にはパネルヒーター・ヒートケーブル・セラミックヒーターランプなどを使用
- サーモスタットで温度管理すると安定しやすい
- 爬虫類は変温動物のため、極端な温度低下は肺炎など命に関わる病気を引き起こす

## 湿度管理

- 適正範囲：40〜60%
- 高湿度では細菌・真菌の繁殖を促し病気のリスクが上がる
- 低湿度（過乾燥）では脱水症状や甲羅の形成不全を引き起こすことがある
- 霧吹きや湿度調整器で調整する。湿度が高い場合は換気を行う
- 砂と腐葉土を混ぜた床材を使うと適度な湿度を保ちやすい

## 温浴

週1〜2回、30〜35℃のぬるま湯で5〜10分温浴させる。

温浴の目的：
- 便秘予防（体が温まることで腸の動きが活発になる）
- 皮膚の保湿
- 水分補給

終了後は体をしっかり拭いて冷えを防ぐ。

## 日常のケア

| 頻度 | 内容 |
|------|------|
| 毎日 | フン・食べ残しの除去、水皿の水交換 |
| 週1回 | 床材の部分交換、照明器具・バスキングライトの状態確認 |
| 月1回 | ケージ全体の洗浄、カビ・雑菌の確認 |

## ハンドリング

- お迎え直後はストレスになるため、ケージ内で自由に動き回り餌を食べるようになってからハンドリングする
- オムツをはかせて室内散歩も可能。バスキングスポット・餌場・水場・隠れ家を整えれば放し飼いも可能
- 発情したオスはメスに噛みつくなどして交尾を迫るため、必要に応じて隔離する
