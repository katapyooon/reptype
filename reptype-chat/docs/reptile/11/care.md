# アカメカブトトカゲ - 飼育方法

## ケージ

- 60cmクラスの水槽でペア飼育が可能
- 非常に臆病なシークレッティブな種のため、隠れ場所を十分に用意する

## 温度・湿度

| 項目 | 目安 |
|------|------|
| 適温 | 22〜27℃ |
| ホットスポット | 不要 |

- 低温に弱いため、冬季は適切な保温が必要
- 高温にも弱いので夏季の過加熱に注意（ただし極端な高温でなければ許容範囲あり）
- 皮膚が露出している部分が多いため、乾燥に弱い。湿度管理が重要
- 多湿系トカゲの飼育に準じ、ケージ内の湿度を高めに維持する

## 床材

- ヤシガラ土など湿度を維持できる素材を使用する
- 落ち葉や倒木を模したシェルターを設置し、自然に近い環境を再現する

## 照明

- 強い紫外線灯やバスキングランプは不要
- フルスペクトルランプ程度で十分

## 容器内レイアウト

- シェルター（隠れ家）を必ず設置する
- 大きめの水入れを設置する（水辺を好み、水浴びをすることがある）
- 落ち葉・流木・コルクなどで隠れ場所を多く作る

## 基本的な世話

多湿系トカゲの飼育に準ずる。定期的に霧吹きを行い、湿度を維持する。
