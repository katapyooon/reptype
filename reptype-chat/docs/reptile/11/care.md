# アカメカブトトカゲ - 飼育方法

生息地はニューギニア島（インドネシア）・パプアニューギニアの熱帯雨林。渓流周辺のやや涼しい環境に暮らす夜行性のトカゲ。臆病な性格と乾燥への弱さから飼育難易度はやや高いが、環境を正しく整えれば安定した飼育が可能。

## ケージ

- 最低でも45〜60cmケージを用意する。成体には60cmでも小さく感じることがあり、余裕があれば90cmが理想
- 非常に臆病なため、隠れ場所を十分に用意することが最優先
- ケージ内に岩や流木を配置すると立体活動も行うため、ある程度レイアウトしてあげるとよい
- 多頭飼育の場合は60cm以上を用意し、個体数以上のシェルターを必ず設置する

## 温度管理

| エリア | 温度目安 |
|--------|---------|
| ホットスポット | 28〜30℃ |
| ケージ内（通常） | 25〜27℃ |
| 夜間 | 22℃前後 |

- 低温に弱いため冬季は適切な保温が必要。高温にも弱いため夏季の過加熱に注意
- **温度勾配を必ず設ける**こと。ホットスポットと涼しいエリアを作りカメが自分で移動できるようにする
- **サーモスタットは必須**。温度が上がりすぎないよう自動制御する

### 保温器具

- **パネルヒーター**：ケージ底面または側面に設置して保温する
- **保温球（25W程度）**：バスキングスポットを作りたい場合に使用。ケージ全体が高温にならないよう注意

## 湿度管理

- **70%以上を維持することが必須**。皮膚が非常に薄く乾燥に極めて弱い。60%台でも本種には乾燥環境となる
- 霧吹きを定期的に行い、湿度計でこまめに確認する
- 通気性も重要。蒸れが続くと皮膚病のリスクが高まるため、換気できるケージを使用する

## 床材

湿度を保てるかどうかが選択の最重要ポイント。

| 床材 | 特徴 |
|------|------|
| テラリウムソイル | 湿度維持に優れ管理しやすい。糞の処理も容易で**最もおすすめ** |
| ヤシガラ土 | 自然に近く穴掘りも可能。乾燥すると土埃が出やすい |
| 水苔 | 湿度維持には最良だが、ケージ全体に使うと糞の処理が困難。タッパーに入れて部分使用が現実的 |
| ウッドチップ | 汎用性はあるが本種の高湿度維持には不十分なためあまり推奨しない |

床材は月1回を目安に全体を取り出して清掃する。

## シェルター

臆病な本種には**必ず複数設置**する。安心できる環境が安定した飼育の鍵。

- ウェットシェルター（内部が湿った状態を保てるもの）が特に有効。水浴びを好む性質にも対応できる
- シェルターは**大きすぎず、全身が入って少し窮屈なくらいのサイズ**が安心感を与える
- **涼しいエリアと暖かいエリアの両方**にシェルターを設置し、個体が自分で快適な場所を選べるようにする
- 落ち葉・流木・コルクなどで隠れ場所を多く作ることも有効

## 照明

- 強い紫外線灯やバスキングランプは不要
- **フルスペクトルランプ**程度で十分。カルシウム代謝サポートのため弱めのUVBライトを設置してもよい
- UVBを使用する場合は照射距離と時間に注意し、紫外線量が強すぎないようにする
- 日中のみ点灯し、夜間は消灯して昼夜リズムを作る

## 水入れ

- 水辺を好み水浴びをすることがあるため、全身が浸かれる大きめの水入れを設置する
- 水は清潔を保つため定期的に交換する

## 基本的な世話

定期的に霧吹きを行い湿度を維持する。定期的な温浴で皮膚の汚れを取り除くことも皮膚病予防に有効。

## 注意すべき病気

**皮膚病**：高湿度環境が続くと真菌（カビ）感染症のリスクが高まる。また温度が低すぎると脱皮不全を起こし、古い皮膚が血行を妨げて指先・尾先の壊死につながることがある。

**くる病**：カルシウム・ビタミンD3不足で起きる。エサにはカルシウム＆ビタミンD3サプリメントをダスティングして与える。

**拒食・突然死**：臆病な性格のため、環境変化やストレスで拒食になりやすい。湿度・温度の不備や寄生虫感染が突然死の原因になることもある。適切な環境の維持が最大の予防策。
