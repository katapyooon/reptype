# アカメカブトトカゲ（type_id: 11）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Tribolonotus gracilis* |
| 和名 | アカメカブトトカゲ（別名：メベニカブトトカゲ） |
| 英名 | Orange-Eyed Crocodile Skink |
| 分類 | 爬虫綱 トカゲ目 トカゲ科 カブトトカゲ属 |

## 分布

インドネシア（ニューギニア島）、パプアニューギニア（ニューギニア島）

## 形態

- 全長：15〜20cm（最大193mm）
- 成体は目の周りが赤やオレンジ色になる（和名・英名の由来）
- 胴体や四肢の鱗は大型で、ワニや恐竜を彷彿とさせるゴツゴツした外観
- 皮膚が露出している部分も多い
- 尾にはやや鉤状になった鱗が並ぶ
- 発声器官を持ち、鳴くことができる（爬虫類では珍しい特徴）

## 生態

- 熱帯雨林の林床に生息する
- 夜行性で、昼間は落ち葉や倒木の下などで休む（昼行性という情報もある）
- 水辺を好み、水中で水浴びすることがある

## 食性

動物食。昆虫類・節足動物・カエルの幼生・小魚などを食べる。雑食性という情報もある。

## 性格・行動

- 臆病な性格で、飼育下においても人前に姿を現すことは少なく、基本的には隠れている
- 天敵が現れた際は硬直して動かなくなる
- 威嚇の際に声帯から鳴き声を発することがある
- ストレス等で尾の自切をすることがある

## 繁殖

卵生。1回に1〜3個の卵を産み、孵化まで約60日かかる。産卵された卵はメス親が世話をし、孵化後も数週間はメス親が幼体の世話をする。

## 人間との関係

- パプアニューギニアの切手のモチーフとなったことがある
- ペットとして飼育されることもあり、日本にも輸入されている
- 以前はカブトトカゲ属の別種モトイカブトトカゲ（ニューギニアカブトトカゲ）の名前で紹介・流通していた
- かつては輸送状態が悪く、状態を崩している個体が多かったことから飼育が難しい種とされた
- 近年は輸送状態も改善されたため、以前に比べて飼育は難しくないとされる
- 高温より低温に弱い。皮膚が露出しているため乾燥にも弱い
- 飼育下繁殖は確立されていないものの、日本国内での飼育下繁殖例がある
