# アカメカブトトカゲ - 餌・給餌

## 主食

基本的に昆虫食。

- コオロギ（主食として最適）
- ミールワーム
- その他の昆虫類・節足動物

自然下ではカエルの幼生・小魚なども捕食する。雑食性という情報もあり、イグアナフードなどの人工飼料を食べる場合もある。

## 給餌頻度

個体の状態・サイズに応じて調整する。食欲は臆病な性格から飼育下では確認しにくい場合があるため、夜間など落ち着いた時間帯に給餌するとよい。

## サプリメント

カルシウム・ビタミンなどの添加剤は必須。餌にダスティングして与える。

**ダスティング**：エサにカルシウム剤などをまぶして与えること。

## 水分補給

大きめの水入れを設置し、自由に水浴びや飲水ができる環境を整える。水辺を好む種のため、清潔な水を常時用意する。
