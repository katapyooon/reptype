# ヒナタヨロイトカゲ（type_id: 16）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Cordylus tropidosternum* (Cope, 1869) |
| 和名 | ヒナタヨロイトカゲ、ネッタイヨロイトカゲ、トロピクスヨロイトカゲ |
| 英名 | Tropical girdled lizard |
| 分類 | 爬虫綱 有鱗目 ヨロイトカゲ科 ヨロイトカゲ属 |
| 全長 | 60〜75mm（最大88mm）※Wikiでは全長15cmと記載 |

## 分布

アフリカ南東部。亜種により分布が異なる。

| 亜種 | 分布地域 |
|------|---------|
| *C. t. tropidosternum*（基亜種） | ケニア、コンゴ民主共和国、ザンビア、ジンバブエ、タンザニア、マラウィ、モザンビーク |
| *C. t. jonesii*（ジョーンズヨロイトカゲ） | ジンバブエ、ボツワナ東部、南アフリカ共和国、モザンビーク |

## 形態

- 全長15cm。体形は扁平
- 体色は褐色や黄褐色で、不規則な暗色の横帯が数本入ることもある
- 腹面は白色から黄褐色
- 胴体の鱗には筋状の盛り上がり（キール）が見られる
- 尾には棘状の鱗が環状に並ぶのが特徴的

### ジョーンズヨロイトカゲ（*C. t. jonesii*）
- 基亜種に比べると小型で、体形は太い
- 独立種とする説もある

## 生態

- サバナや低木林などに生息
- 倒木の下・空洞になった倒木の中・木の切り株の根もとの下などに隠れる
- 昼行性。乾燥した環境を好む
- 食性は動物食で、昆虫類・節足動物（クモ類など）を食べる
- 繁殖形態は卵胎生（胎生）。1回に2〜4匹の幼体を出産する

## 寿命

詳細は不明。飼育下での長期飼育例は少なく、繁殖法も未確立。

## 外見・触感

- 小型だが尾の棘状の鱗はヨロイトカゲらしさを実感できる大きさ
- 国内流通個体には体色に赤みが強い個体も混じることがあり、個体差か別種・別亜種かは不明

## 性格・行動

- 比較的飼育しやすい乾燥系トカゲ
- 飼育環境を整えれば初心者にも向く

## 人間との関係

- ペットとして飼育され、日本にも輸入されている
- 国内で最も多く流通するヨロイトカゲ。季節により大量輸入されるため比較的安価
- 主に基亜種が流通し、亜種ジョーンズヨロイトカゲの流通はまれ
- 岩や流木などを隠れ家に配置したテラリウムで飼育される
- 飼育下での繁殖法は未だ確立していないものの、繁殖例はある
- 雌雄判別がハッキリしないことも多く、国内CBはあまり出回っていない
