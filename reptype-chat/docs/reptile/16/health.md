# ヒナタヨロイトカゲ - 健康管理

## よくある病気・症状

### クル病（代謝性骨疾患 / MBD）

紫外線不足・カルシウム不足・ビタミンD3不足が原因。乾燥系・昼行性トカゲに多い。

**症状：**
- 四肢・背骨の変形
- 顎がやわらかくなる
- 動きが鈍くなる・食欲低下

**予防：**
- 強い紫外線ライト（メタハラ推奨）を適切な距離で照射する
- 毎回の給餌でカルシウムパウダーをダスティングする
- 可能であれば屋外での直射日光浴を行う

### 脱水・食欲不振

温度が低すぎる、または紫外線が不足すると活性が落ち、食欲不振になりやすい。
毎朝の霧吹きで水分補給の機会を確保することが重要。

### 消化不良・腸閉塞

- 床材の砂の誤飲による腸閉塞に注意
- 体温が下がっている状態での給餌は消化不良の原因になる
- 餌を与える前にバスキングで十分に体温を上げさせる

## 脱皮について

部分脱皮。全身が一度に剥がれるのではなく、部位ごとに段階的に脱げる。

- 脱皮不全は湿度不足・栄養不足が原因になることがある
- 毎朝の霧吹きは脱皮補助にも有効
- 無理に脱皮殻を剥がさない（皮膚を傷つける恐れがある）

## 動物病院に行く目安

- 食欲不振が1週間以上続く
- 四肢・背骨の変形が見られる
- 動きが著しく鈍くなった・ぐったりしている
- 脱皮不全が繰り返し起こる
- 体重が急激に減少している
