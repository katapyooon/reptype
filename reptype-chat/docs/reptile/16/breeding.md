# ヒナタヨロイトカゲ - 繁殖

## 繁殖様式

卵胎生（胎生）。卵を体内で孵化させ、幼体の状態で出産する。

## 繁殖適齢期

詳細は不明。飼育下での繁殖法は未だ確立していないが、飼育下での繁殖例は報告されている。

## クーリング

野生では季節変化のある環境に生息しているため、繁殖を促す場合は温度を一定期間下げるクーリングが有効な可能性がある。ただし確立した方法はなく、個体の状態を見ながら慎重に行うこと。

## 出産・幼体について

- **出産時期**：夏（野生下の記録に基づく）
- **産仔数**：1回に2〜4匹の幼体を出産
- **幼体サイズ**：全長60〜70mm程度

## 幼体の育て方

- 幼体は成体と同様に昆虫食
- 小型の昆虫（小コオロギ・ショウジョウバエなど）から慣らす
- 成体との同居は捕食・ストレスの原因になるため避ける
- 飼育下での繁殖例は少なく、情報が限られているため国内の繁殖例を参考にしながら対応する

## 注意事項

雌雄判別が難しいとされており、ペアリングが容易でないことが国内での繁殖例が少ない一因と考えられる。
