# ヒナタヨロイトカゲ - 飼育方法

## ケージ

- **推奨サイズ**：60cmクラス以上。床面積がなるべく広いものを選ぶ
- 活発に動き回るため、広いほど行動を観察しやすくストレス軽減になる

## 温度管理

変温動物のため、ケージ内に必ず**温度勾配**を作ること。

| エリア | 温度目安 |
|--------|---------|
| バスキングスポット | 40℃以上（ケージが広い場合は50℃程度） |
| 常温エリア（クールスポット） | 25℃程度 |
| 夜間 | 保温器具をすべて停止 |

- 夜間はすべての保温を止め、自然に近い昼夜の温度差をつける
- 温度勾配があることで、個体が自分で体温調節できる環境を作る

## 照明

強い紫外線環境が必要。

- **UVライト**：メタルハライドランプ（メタハラ）または爬虫類用の強い紫外線灯が必要
- **バスキングランプ**：必須。ホットスポットで体を温められるよう設置する
- **直射日光**：可能であれば屋外での日光浴を行う（最も効果的）

照明時間は昼間のみ。夜間は消灯して昼夜リズムを作る。

## 床材

- 砂をやや厚めに敷く
- 乾燥した砂地環境を再現することが重要

## 容器内レイアウト

- 流木・岩・ブロックなどでシェルターを兼ねたバスキングスポットを作る
- 隠れ家は複数箇所用意すると安心
- 小さめで倒れない水入れも設置する

## 湿度・霧吹き

- 基本は乾燥系の飼育に準ずる
- 毎朝霧吹きを行い、水分補給の機会を与える
- ケージ全体は乾燥させたまま、局所的に湿度をつくるイメージ

## 基本的な世話

- 乾燥系トカゲの飼育に準ずる
- 毎朝霧吹き後に餌やりのタイミングをとるとよい
- 糞はこまめに取り除いて衛生を保つ
