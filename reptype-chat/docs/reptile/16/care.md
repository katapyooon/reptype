# ヒナタヨロイトカゲ - 飼育方法

アフリカ南東部（ケニア・タンザニア・ザンビアなど）の乾燥したサバンナや低木林・岩場に生息する小型トカゲ。鎧状の鱗と棘状の突起が特徴的で、見た目のインパクトとは裏腹に温厚で臆病な性格。多頭飼育にも比較的適している。

## ケージ

- **推奨サイズ**：最低60×40×40cm、理想は90×45×45cm。床面積は広いほどよい
- 活発に動き回るため、広いほど行動を観察しやすくストレス軽減になる
- 通気性確保のためメッシュ蓋を推奨

## 温度管理

変温動物のため、ケージ内に必ず**温度勾配**を作ること。

| エリア | 温度目安 |
|--------|---------|
| バスキングスポット | 32〜40℃（広いケージでは40℃以上も可） |
| 常温エリア（クールスポット） | 24〜30℃ |
| 夜間 | 20〜24℃ |

- 温度勾配があることで、個体が自分で体温調節できる環境を作る
- 夜間は昼夜の温度差をつけることが重要だが、20℃を下回る場合は保温器具で補う

## 保温器具

- **バスキングランプ**：必須。ホットスポットで体を温められるよう設置する
- **パネルヒーター**：夜間や冬季にケージ内温度が下がりすぎる場合に補助的に使用する
- **セラミックヒーター**：光を出さずに保温できるため夜間の補助保温に適している
- **サーモスタット**：温度の自動管理に使用し、過加熱・過冷却を防ぐ

## 照明

強い紫外線環境が必要。

- **UVライト**：メタルハライドランプ（メタハラ）または爬虫類用の強い紫外線灯が必要
- **バスキングランプ**：ホットスポットを作るために設置
- **直射日光**：可能であれば屋外での日光浴を行う（最も効果的）

照明時間は1日12時間を目安に昼間のみ点灯し、夜間は消灯して昼夜リズムを作る。

## 床材

- 砂やココナッツファイバー、ウッドチップなどをやや厚めに敷く
- 乾燥した砂地環境を再現することが重要

## シェルター・レイアウト

倒木や岩の隙間を好む習性に合わせてシェルターを必ず設置する。

- 流木・岩・植木鉢・ブロックなどでシェルターを兼ねたバスキングスポットを作る
- **隠れ家は複数箇所**用意するとストレス軽減につながる
- 立体的なレイアウトで高さを出すと自然に近い環境を再現できる
- 小さめで倒れない水入れも設置する（毎日交換して清潔を保つ）

## 湿度・霧吹き

- 適正湿度：40〜60%
- 基本は乾燥系の飼育に準ずる
- 毎朝霧吹きを行い、水分補給の機会を与える
- ケージ全体は乾燥させたまま、局所的に湿度をつくるイメージ

## 基本的な世話

- 毎朝霧吹き後に餌やりのタイミングをとるとよい
- 糞はこまめに取り除いて衛生を保つ
- 床材とケージ内は週1回の部分清掃、月1回の全体清掃を目安に行う
- 体調の異変（食欲不振・脱皮不良・元気消失）が見られたら爬虫類専門の動物病院に相談する
