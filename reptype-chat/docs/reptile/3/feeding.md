# クレステッドゲッコー - 餌・給餌

## 食性
動物食傾向の強い雑食。野生下では昆虫、節足動物、果実、花の蜜なども食べる。植物質を摂取するのは、獲物の少ない環境に対する適応と考えられている。

## 主食
飼育下では以下のものを与える。
- 昆虫類（コオロギ、デュビアなど）
- 果実
- 昆虫ゼリー
- クレステッドゲッコー専用の人工飼料（CGD：Crested Gecko Diet）

専用の人工飼料は栄養バランスが取れており、主食として利用できる。

## 給餌頻度
成体は2〜3日に1回が目安。幼体はより高頻度で与える。

## サプリメント
昆虫を与える場合はカルシウムやビタミンDのダスティングを行う。専用人工飼料のみで飼育する場合は別途サプリメントが不要な場合も多い。

## 水分補給
テラリウムの壁面に霧吹きで水を吹きかけることで水分を摂取する。水入れを設置してもよい。
