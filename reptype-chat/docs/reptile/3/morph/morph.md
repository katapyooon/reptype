# クレステッドゲッコー - モルフ一覧

## 概要
飼育の歴史こそ短いものの、赤や黄色など様々な体色をした個体を掛け合わせたモルフが産出されている。体色には変異があり、多彩な品種が流通している。

## 代表的なモルフ（日本流通名）
| 流通名 | 特徴 |
|---|---|
| オレンジ | 鮮やかなオレンジ色の体色 |
| タイガー | 縞模様が入る |
| ダルメシアン | 斑点模様が入る |
| ファイア | 赤みがかった明るい体色 |
| レッド | 赤色の体色 |

## 遺伝情報
クレステッドゲッコーのモルフは体色・模様の組み合わせによるもので、ヒョウモントカゲモドキほど単純メンデル遺伝で整理されていない。品種改良の歴史が進むにつれ、より多様な個体が生み出されている。
