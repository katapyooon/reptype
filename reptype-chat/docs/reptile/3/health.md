# クレステッドゲッコー - 健康管理

## よくある症状・注意事項

### 尾の自切
尾を自切した場合、基部に玉状の組織が形成される。他のヤモリと異なり、元のような長い尾には再生しないため注意が必要。

### 土食い（異食）
過度にストレスがかかると、発作的に口を開けて暴れまわり、ヤシガラなどの床材を口にする「土食い」を行うことがある。体内に土が詰まると死亡する場合があるため、ストレス要因の排除が重要。

## 動物病院に行く目安
- 食欲不振が長期間続く場合
- 脱皮不全が起きている場合
- 異食後に元気がない・排泄がない場合
- 外傷や腫れがある場合

## 脱皮について
定期的に脱皮を行う。湿度が低いと脱皮不全を起こしやすいため、脱皮前後は十分に加湿する。幼体は最初の脱皮まで餌を食べないのが正常な行動。
