# フトアゴヒゲトカゲ - 餌・給餌

雑食性。野菜・葉物（植物質）と昆虫（動物質）をバランスよく与える。
成長段階に応じて比率を調整することが重要。

## 成長段階別の給餌目安

| 時期 | 体長 | 昆虫：野菜：人工フード | 頻度 |
|------|------|----------------------|------|
| ベビー期（〜生後3〜4か月） | 〜20cm | 8：1：1（昆虫メイン） | 1日2回 |
| ヤング期（生後4〜8か月） | 20〜30cm | 6：2：2（昆虫やや多め） | 1日1回 |
| ヤングアダルト期（生後8か月〜1歳） | 30〜40cm | 3：4：3（野菜中心へ移行） | 1日おき〜1日1回 |
| アダルト期（1歳〜） | 40〜50cm | 0：7：3（野菜メイン） | 3日に1回 |

## 給餌のコツ

- **与える順番**：野菜 → 昆虫の順。先に昆虫を与えると野菜を食べなくなる
- **ベビー〜ヤング期**：欲しがるだけ与えてOK。人工フードはケージ内に常設しておくと慣れやすい
- **アダルト期**：肥満予防のため体重・排便のペースを確認しながら給餌頻度を調整する
- 野菜を食べないときは昆虫パウダーをふりかけて嗜好性を上げると効果的

## サプリメント

- **カルシウムパウダー**：野菜や昆虫にダスティング（毎回）
- **ビタミンD3配合カルシウム剤**：週1回程度。毎日与えると過剰摂取になるため注意

ビタミンD3はカルシウム吸収・骨格形成に不可欠だが、体内に蓄積しやすいため頻度を守ること。

## 水分補給

- 飲み水は毎日交換して用意する
- フトアゴヒゲトカゲは水入れの水面を「水」と認識しにくいことがある
- 認識しない場合：指で水面をピチャピチャと動かして教える、霧吹きで口元に吹きかける、スポイトで与えるなどを試す
