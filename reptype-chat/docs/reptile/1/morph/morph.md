# フトアゴヒゲトカゲ - モルフ一覧

体色や模様の違いによって「モルフ」と呼ばれる多くの品種が存在する。
野性的なノーマルから希少性の高いハイポゼロまで多種多様。

## 代表的なモルフと価格目安

| モルフ名 | 価格目安 | 特徴 |
|---------|---------|------|
| ノーマル（ワイルドタイプ） | 1〜2万円 | グレー〜茶褐色。最も入手しやすく丈夫で飼育しやすい基本モルフ |
| レッド | 2〜4万円 | 赤みの強い個体は人気が高く、発色の良さで価格が変動 |
| イエロー | 2〜4万円 | 明るい黄色の体色。初心者から上級者まで人気 |
| ハイポ（ハイポメラニスティック） | 3〜6万円 | 黒色素が少なく透明感のある体色が特徴 |
| ゼロ | 6〜10万円 | 模様がほとんどない白系モルフ。希少性が高い |
| ハイポゼロ | 8〜15万円以上 | ハイポとゼロを併せ持つ希少モルフ。発色の美しさから特に人気 |

## その他のモルフ

- **トランス**：鱗に透け感があり、黒目がちな瞳が特徴
- **レザーバック**：体表のトゲが少なく滑らかな質感
- **ブラッドレッド**：深紅の体色を持つ希少モルフ
- **パラドックス**：独特の斑点模様が入る。希少性が高く10万円を超えることも
