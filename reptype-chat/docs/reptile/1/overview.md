# フトアゴヒゲトカゲ（type_id: 1）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Pogona vitticeps* (Ahl, 1926) |
| 和名 | フトアゴヒゲ、フトアゴヒゲトカゲ |
| 英名 | Central bearded dragon |
| 分類 | 爬虫綱 有鱗目 アガマ科 アゴヒゲトカゲ属 |

## 分布

オーストラリア（クイーンズランド州、ニューサウスウェールズ州、ビクトリア州、南オーストラリア州、ノーザンテリトリー）

## 形態

- 最大全長：49cm（頭胴長23cm・尾長26cm）
- 体形は太く、後頭部の棘状鱗が横一列に並ぶ
- 体側面の刺状鱗は規則的でほぼ1列
- 口内の色彩はピンク色の個体が多い

## 生態

- 森林から砂漠にかけての様々な環境に生息
- 半樹上棲・昼行性
- 危険を感じると口を開け、喉の皮膚を広げて威嚇する
- 頭や前足を振って他のオスやメスとコミュニケーションをとる
- 雑食性（主に昆虫を食べるが植物質なども食べる）

## 人間との関係

- アゴヒゲトカゲ属内で最も流通量が多いペット爬虫類
- 流通するのは飼育下繁殖個体のみ（野生個体は流通しない）
- 人間によく懐き、大人しい性格で初心者にも飼いやすい
