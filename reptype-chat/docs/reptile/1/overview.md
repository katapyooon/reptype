# フトアゴヒゲトカゲ（type_id: 1）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Pogona vitticeps* (Ahl, 1926) |
| 和名 | フトアゴヒゲ、フトアゴヒゲトカゲ |
| 英名 | Central bearded dragon |
| 分類 | 爬虫綱 有鱗目 アガマ科 アゴヒゲトカゲ属 |

## 分布

オーストラリア（クイーンズランド州、ニューサウスウェールズ州、ビクトリア州、南オーストラリア州、ノーザンテリトリー）

## 形態

- 最大全長：49cm（頭胴長23cm・尾長26cm）
- 体形は太く、後頭部の棘状鱗が横一列に並ぶ
- 体側面の刺状鱗は規則的でほぼ1列
- 口内の色彩はピンク色の個体が多い

## 生態

- 森林から砂漠にかけての様々な環境に生息
- 半樹上棲・昼行性
- 危険を感じると口を開け、喉の皮膚を広げて威嚇する
- 頭や前足を振って他のオスやメスとコミュニケーションをとる
- 雑食性（主に昆虫を食べるが植物質なども食べる）

## 寿命

平均寿命はおよそ5〜10年。飼育環境が整っていると15年近く生きる個体もいる。
爬虫類の中では比較的長生きする種類。

## 外見・触感

- 平たい体つきと力強い四肢が特徴
- のど元のトゲ状の鱗が名前の由来
- 体の皮膚はゴツゴツした見た目だが触るとゴムのような弾力がある
- トゲの感触は柔らかいがザラっとしており、アゴ部分はふわふわ

## 性格・行動

- 昼行性で朝〜夕方に活発。人間の生活時間帯に近い
- 人間によく懐き、ハンドリングを受け入れる個体が多い
- カラーモルフ（体色の品種）が豊富で個体ごとに異なる美しさを楽しめる

## 特有の動作

| 動作 | 意味 |
|------|------|
| **ボビング**（頭を上下に振る） | 威嚇、またはオスのメスへの求愛行動。アゴが黒くなることもある |
| **アームウェービング**（腕を円を描くように回す） | 敵意がないことを示す服従・挨拶行動。メスがオスの求愛に応じる際にも |
| **アゴ体操**（アゴを膨らませ凹ませを繰り返す） | 生理的な伸び、体温調節、ストレス発散など諸説あり |

## 人間との関係

- アゴヒゲトカゲ属内で最も流通量が多いペット爬虫類
- 流通するのは飼育下繁殖個体のみ（野生個体は流通しない）
- 人間によく懐き、大人しい性格で初心者にも飼いやすい
- 購入は動物愛護法により対面販売のみ（ペットショップ・爬虫類イベント）
