# フトアゴヒゲトカゲ - 飼育方法

## ケージ

- **推奨サイズ**：横幅90cm以上（幅90×奥行45×高さ45cmが標準）
- スペースがあれば120cmがより理想。広いほど運動量が確保でき肥満予防・ストレス軽減になる
- 最小60cmでも飼育可能だが、成体には手狭。その場合は部屋んぽで運動量を補う

成長に伴い体長が変化するため、体長に合わせてケージサイズを選ぶこと。

| 成長段階 | 体長の目安 |
|----------|-----------|
| ベビー | ～20cm |
| ヤングアダルト | 20～40cm |
| アダルト | 40cm～（最大50cm程度） |

## 温度管理

変温動物のため、ケージ内に必ず**温度勾配**を作ること。

| エリア | 温度目安 |
|--------|---------|
| バスキングスポット | 35〜40℃ |
| ケージ中央 | 30℃前後 |
| クールスポット | 25〜28℃ |
| 夜間（保温） | 23〜25℃ |

- 夜間はバスキングライトを消灯し、保温器具をサーモスタットで管理する
- 温湿度計はケージ内の暖かい場所と涼しい場所の2か所に設置する
- 湿度の目安は60〜70%

## 保温器具

寒さ対策には以下の3種類を組み合わせると安心。

- **全体保温球**：ケージ全体の温度を底上げする
- **バスキングランプ**：局所的に熱を当て、バスキングスポット（日向ぼっこする場所）を作る
- **パネルヒーター**：ケージ床面の1/3〜1/2のサイズのものをケージ底面に敷く

全体保温球とバスキングランプを組み合わせてバスキングスポットを設け、個体が自分で体温調節できる環境を整える。

## 照明

3種類のライトを組み合わせる：

- **バスキングライト**：熱を発生させ体温上昇を助ける
- **UVライト（UVB）**：ビタミンD3合成・カルシウム吸収に不可欠。照射時間は1日12時間が目安
- **可視光線**：昼夜リズムを作る。UVライトの直視防止にも重要

**UVライト設置の注意点：**
- ガラス・プラスチック越しだと紫外線がほぼ遮断されるため、ケージ内部に設置する
- 爬虫類はUVライトの光を認識しにくく直視しやすいため、照射角度に注意する

## 床材

水に濡れても乾きやすいものを選ぶ。

- **サンドタイプ**：砂漠の環境を再現。消臭・保温効果もある
- **ソイルタイプ**：排泄物がまとまりやすく管理しやすい。消臭・保温効果もある
- **ペットシーツ・キッチンペーパー**：慣れないうちは扱いやすい

砂や細かいチップは誤飲すると消化管で詰まる危険があるため注意。フンはその都度取り除いて衛生を保つ。

## シェルター

紫外線の過剰照射を防ぐためにシェルターを設置する。

- 個体が身体を乗せて保温球で温まれる程度のサイズが目安
- 乗った際に保温球で火傷しない高さのものを選ぶ
- ケージ内の温度勾配に合わせてクールスポット側にも隠れ場所を用意するとよい

## ハンドリング

- **上から掴まない**：天敵（鳥）の動きに似ており本能的に恐怖を感じる
- **下から支える**：お腹の下に手を入れ、もう片方でお尻〜尾の付け根を支える
- **指先に顔を向ける**：進めない方向に顔が来るようにすると落ち着きやすい
- フトアゴヒゲトカゲには**自切（尻尾を切り離す習性）はない**

## 温浴

週に1〜3回、35℃程度のお湯に入れて温浴させる。

温浴の目的：
- 水分補給
- 体を清潔に保つ
- 便秘予防
- 脱皮の促進

水深はお腹が少し浸る程度から始め、無理強いしないようにする。

## 運動・部屋んぽ

ケージ内だけでは運動不足になりやすいため、定期的に実施する：

- **ハンティング運動**：生き餌（コオロギなど）をケージ内に放して追いかけさせる（1〜2匹で切り上げる）
- **部屋んぽ**：室温22〜28℃に調整の上、誤飲物・電源コードなど危険物を除去してから実施
- **屋外散歩**：暖かい季節に日光浴を兼ねて。ハーネス必須、外敵・逃走に注意

## 飼育上の注意

- 幼体は互いに噛みつく・共食いするため単独飼育が基本
- オス同士は激しく争うため複数飼育は避ける
