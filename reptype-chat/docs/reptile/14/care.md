# アカメアマガエル - 飼育方法

## ケージ

高さと通気性を重視したケースを使用する。樹上棲のため、**高さのあるケージ**が望ましい。

- 観葉植物を鉢ごと入れ、木の枝や流木を組んで活動場所や隠れ家を作る
- タッパーなどの水容器を設置する（地表・樹上に全身が浸かれる程度の大きさ）

## 温度管理

| 状態 | 温度目安 |
|------|---------|
| 昼間 | 24〜29℃ |
| 夜間 | 18〜21℃ |
| 冬場（保温時） | 20〜24℃ |

- 極度の低温には弱いため、冬季には暖房器具を使用するか、ケージを温室に収納する

## 湿度管理

- 夜間や照明を消した際に霧吹きで湿度を上げる（毎日実施）
- 常時湿らせすぎにならないよう注意する
- 湿度を上げることで活動が活発になるため、給餌は霧吹きをしてから行うのが望ましい

## 照明

特に必要なし。

## 床材

ヤシガラ土・腐葉土・水苔などを組み合わせて、湿度をコントロールできる素材を**厚めに**敷く。

- メンテナンス性を重視する場合はキッチンペーパーも使用可能だが、汚れが溜まりやすいためこまめに交換する

## セッティング例

1. ヤシガラ土や腐葉土などの床材を厚めに敷く
2. 観葉植物（鉢ごと）と木の枝・流木を配置して立体的な活動スペースを作る
3. 全身が浸かれる水容器を設置する
4. 温湿度計を設置してケージ内の環境を確認する
5. 毎晩霧吹きで湿度を上げる
