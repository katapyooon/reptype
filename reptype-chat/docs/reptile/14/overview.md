# アカメアマガエル（type_id: 14）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Agalychnis callidryas* |
| 和名 | アカメアマガエル（赤目雨蛙） |
| 英名 | Red-eyed Tree Frog |
| 分類 | 両生綱 無尾目 アマガエル科 アカメアマガエル属 |
| 原産地 | 中米（メキシコ南部〜パナマ） |
| 全長 | 3〜7cm（オス最大5.6cm、メス最大7cm） |

## 分布

グアテマラ、コスタリカ、ニカラグア、パナマ、ベリーズ、ホンジュラス、メキシコ南部。

## 形態

- 体長3〜7cm。メスの方がオスより大型で、オスは最大でも5.6cm
- 背面は緑色
- 体側面は青や紫と黄色・白の横縞模様
- 後肢の内側は青い
- 指趾の色彩はオレンジ色
- 背面に白い斑点が入る個体もあり、地域・環境により色彩に変異がある

眼は大きく、瞳孔は縦長。虹彩が赤いことが和名・英名（Red-eyed）の由来。種小名 *callidryas* は「美しい木の妖精」を意味する。

## 分類

アマガエル科アカメアマガエル属に分類される。メキシコとグアテマラの個体群を亜種 *A. c. taylori* として分割する説もある。

## 生態

熱帯雨林や二次林に生息する樹上棲のカエル。夜行性で、昼間は葉の上で目を閉じ四肢を縮こまらせてうつ伏せになって休む。この姿勢により表面積を減らして乾燥を防ぎ、背面の緑色のみが見える状態になるため葉と区別しにくくなる。

敵に襲われた際は目を開けてジャンプし、鮮やかな四肢と体側の斑紋で敵を驚かせると言われている。

**食性：** 動物食性。昆虫類・節足動物・小型のカエルなどを捕食する。

**繁殖：** 卵生。雨季に繁殖が行われ、水面の上に垂れ下がる葉に1回につき10〜70個のゼラチン質に覆われた卵を産みつける。孵化した幼生はそのまま下の水場に落ち、水中生活を始める。産卵は数クラッチに分けて行われ、産卵ごとにメスは水に入り体内に水分を補給する。

## 人間との関係

その特徴的な形態からデザインのモチーフとして多く用いられる。カエル関係の書籍の表紙などに使われることも多い。

ペットとして飼育されることもあり、日本にも輸入されている。流通量は比較的多く、野生個体と飼育下繁殖個体（CB個体）が共に流通する。かつては輸送状態の悪さから体調を崩している個体が多く飼育の難しい種とされていたが、輸送状態の改善や CB 個体の流通に伴い飼育難易度は以前と比べて下がっている。
