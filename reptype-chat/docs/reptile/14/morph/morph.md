# アカメアマガエル - 色彩変異・地域変異

## 基本的な色彩

アカメアマガエルは地域や環境により色彩に変異がある。

- 背面は緑色が基本
- 背面に白い斑点が入る個体もいる
- 体側面の横縞は青・紫・黄色・白など個体差がある

## 亜種・地域変異

メキシコとグアテマラの個体群を亜種 *Agalychnis callidryas taylori* として分割する説がある（原名亜種との区別については諸説あり）。

## 流通について

コーンスネーク等と異なり、アカメアマガエルはカラーモルフの固定品種はほとんど存在しない。流通する個体は主に野生型（ノーマル）であり、地域変異による若干の色彩差が見られることがある。

野生個体（WC）と飼育下繁殖個体（CB）の両方が流通しており、CB個体の方が環境への適応力が高く飼育しやすい傾向がある。
