# アカメアマガエル - 餌・給餌

## 主食

**昆虫食性。** やや小さめのコオロギ等を主食とする。

- コオロギ（やや小さめのサイズを選ぶ）
- その他の昆虫類・節足動物

## 給餌方法

- 餌は餌容器に入れて与える
- **給餌前に霧吹きをして湿度を上げる**と、活動が活発になり餌食いがよくなる
- 生きた餌を与える場合は、反撃によるケガを防ぐため、**顎・触角・後肢を潰したり折ってから**与える

## サプリメント

- 餌に事前に野菜等を与えてガットローディング（餌の栄養価を上げる）を行う
- 給餌前に餌虫へサプリメント（カルシウム剤・ビタミン剤等）を振りかけてダスティングする

## 水分補給

- 地表と樹上に全身が浸かれる程度の水容器を用意する
- 毎晩の霧吹きでも水分を補給できる環境を維持する
