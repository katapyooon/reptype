# アカメアマガエル - 繁殖

## 繁殖の概要

繁殖形態は卵生。野生下では雨季に繁殖が行われる。飼育下でも雨季を模した環境変化（温度・湿度の変動や降雨シミュレーション）を与えることで繁殖を促すことができる。

## 産卵

- 産卵場所：水面の上に垂れ下がる葉の裏側
- 1クラッチの卵数：10〜70個（ゼラチン質に覆われた卵）
- 産卵は数クラッチに分けて行われる
- **産卵ごとにメスは水に入り体内に水分を補給する**。この水分補給をしないと体内の卵が死滅するため注意が必要

## 卵管理

- 卵は高湿度を保った状態で管理する
- 孵化した幼生はそのまま下の水場に落ちて水中生活を始める

## 幼生・幼体の育て方

- 孵化した幼生（オタマジャクシ）は水中で生活する
- 上陸後は成体と同様に小型の昆虫を与える
- 変態直後は非常にデリケートなため、湿度管理を徹底する
