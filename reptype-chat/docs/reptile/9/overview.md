# ニホンイシガメ（type_id: 9）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Mauremys japonica* (Temminck & Schlegel, 1838) |
| 和名 | ニホンイシガメ（別名：イシガメ、幼体はゼニガメ） |
| 英名 | Japanese pond turtle |
| 分類 | 爬虫綱 カメ目 イシガメ科 イシガメ属 |
| 保全状況 | 準絶滅危惧（NT）環境省レッドリスト、ワシントン条約附属書II（2013年） |

## 分布

日本列島の固有種。本州・四国・九州・隠岐諸島・五島列島・対馬・淡路島・壱岐島・佐渡島・種子島に分布。東北地方の記録は人為分布とされ、分布の北限は関東地方と考えられている。

## 形態

- 最大甲長：21cm（メス）、オスは最大14.5cm
- 椎甲板に断続的なキール（瘤状の盛り上がり）がある
- 背甲の色彩は橙褐色〜暗褐色まで個体変異が大きい
- 背甲と腹甲の橋・腹甲の色彩は黒や暗褐色一色（外縁に黄橙色の斑が入る個体もいる）
- 幼体（ゼニガメ）は甲長2.5〜3.5cm、「銭」のような形に見える
- オスは尾が太く長く、総排泄口が背甲の外側に位置する

## 生態

- 河川・湖沼・池・湿原・水田などに生息し、やや流れのある流水域を好む
- 半水棲で水生傾向が強いが、夏季に複数の水場を陸づたいに移動することもある
- 耐寒性が強く、水温3〜5℃での活動記録もある
- 冬季は水中の穴・石の下・堆積した落ち葉の中で冬眠する
- クサガメに比べて水質のきれいな環境を好む

## 食性

雑食性。魚類・カエルとその卵・昆虫・エビ・カニ・巻貝・ミミズ・動物の死骸・水生植物・陸上植物の葉や果実・藻類など幅広く食べる。水中でも陸上でも採食し、農耕地でカキ・トマト・ウリなどを食べることもある。

## 性格・行動

- やや神経質で臆病な傾向がある（クサガメやミドリガメに比べて）
- 十分に馴れた個体は飼い主を他の人間と区別できるようになる
- オスは低温下でも活動する（メスとの繁殖機会を得るための適応と考えられる）

## 人間との関係

- 江戸時代以前からペットとして飼育されている
- 森林伐採・護岸整備・水質悪化・ペット用乱獲・ミシシッピアカミミガメとの競合・クサガメやハナガメとの交雑による遺伝子汚染・アライグマによる捕食などにより個体数・生息域ともに減少している
- 野生個体・飼育下繁殖個体（CB個体）ともに流通する
- 幼体が「ゼニガメ」の名で流通することがあるが、近年ではクサガメの幼体に「ゼニガメ」の名が使われることが多い
- アクアテラリウムや屋外の池での飼育事例が多い
- 日本固有種であっても採集地以外への放流は生息域の系統を脅かすため、絶対に行わないこと
