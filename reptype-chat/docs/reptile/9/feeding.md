# ニホンイシガメ - 餌・給餌

## 食性

雑食性。野生下では機会があればなんでも食べる。

**食べるもの：** 魚類・カエルとその卵・昆虫・エビ・カニ・ヨコエビ類・陸棲および水棲の巻貝・ミミズ・動物の死骸・水生植物・陸上植物の葉や花・果実・藻類など。水中でも陸上でも採食する。

## 餌の種類

- **配合飼料（カメ用ペレット）**：主食として与えやすく栄養バランスが整っている。炭酸カルシウムの添加が推奨される
- **昆虫・甲殻類**：コオロギ・ミルワーム・エビなど
- **魚類**：小魚・金魚・メダカなど
- **野菜・果物**：水棲植物・葉物野菜・カキ・トマトなど
- **卵殻（鶏卵の殻）**：カルシウム補給として喜んで食べる

餌付きの良い個体が多いが、大型の野生個体は餌付きにくいこともある。

## 給餌量

1回に与える量の目安は**カメの頭部の体積×2**。カメの年齢・気温によって加減する。食べ残しは水質悪化の原因になるため早めに取り除く。

## 給餌頻度

| 時期 | 頻度 |
|------|------|
| ベビー期（生後1年未満） | 1日1〜2回、食べるだけ与えてよい |
| 成体 | 2〜3日に1回（食べすぎは肥満の原因） |

- 定期的に体重を計測し、体重が減っていれば給餌頻度を増やす
- 繁殖を目指す場合、晩夏（産卵前の肥育期）は食べるだけ与えて体力を蓄積させる

## 給餌タイミング

- **午前中**に与えるのが最適。太陽が出た後・バスキングランプ点灯後に行う
- 気温が下がる夕方〜夜に与えると体温低下により消化できず、食物が体内で腐敗するリスクがある
- カメは食後に日光浴・バスキングして体温を上げ消化を促す
- 最低でも消灯3時間前までに給餌を済ませる

## 水草との相性

水槽・池に水草を植える場合、ニホンイシガメが底砂を掘り返す・引っ張るため、十分な厚みの底床や大きな鉢植えが必要。

| 植物 | 食べ具合 | 備考 |
|------|---------|------|
| マツモ | ほぼ食べない | 成長が速く水質安定に有効。根がないため沈めて使用 |
| ミクロソリウム | ほぼ食べない | 丈夫な水性シダ。活着性が強い |
| グリーンミリオフォラム | ときどき食べる | 成長速いが十分根づけば引き抜かれにくい |
| アマゾンフロッグビット | ときどき食べる | 繁殖力が強い浮き草。帰化防止のため屋外河川への放流厳禁 |
| オランダガラシ（クレソン） | よく食べる | 餌として有効。スーパーでも入手できる |
| アヌビアス系 | よく食べる（水中） | 陸上植栽では食害が少ない。根茎にシュウ酸カルシウム含有のため扱い注意 |
| アマゾンソードプラント | よく食べる | 餌として利用される |

## サプリメント

- **炭酸カルシウム**（牡蠣殻粉・カルシウムパウダー）：甲羅や骨格形成のために継続的に補給する
- UVB照射が不十分な場合はビタミンD3配合カルシウム剤を使用する（過剰摂取に注意）
