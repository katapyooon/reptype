# ニシアフリカトカゲモドキ - 飼育方法

西アフリカ原産のヤモリの仲間。乾燥した場所にも生息するが、草や岩の隙間といったやや湿度のある環境を好む。夜行性で、日中の多くはシェルター内で過ごす。成体の全長は最大25cm程度で、省スペースでの飼育が可能。

## ケージ

最低限の目安は**横幅30cm×奥行20cm×高さ15cm**。飛び跳ねたり木登りをする種ではないため高さは低くてよいが、広さはストレス軽減のためなるべく確保する。

- 横幅60cm以上あれば快適に飼育でき、レイアウトの自由度も上がる
- 爬虫類専用ガラスケースが管理しやすくおすすめ

## 床材

地表性のため床材は必須。ガラスやプラスチック素材の底面は足が滑って健康を害するリスクがある。

| 床材 | 特徴 |
|------|------|
| 細かい砂状の床材・パームマット | 野生環境に近い。保湿性もある |
| ペットシーツ・キッチンペーパー | 管理が簡単で衛生的 |
| ヤシガラマット | 保湿性が高くおすすめ。汚れやすくダニ発生に注意 |

粒が大きい床材（ハスクチップなど）は誤飲すると腸閉塞になる可能性があるため避ける。

## シェルター

夜行性のため日中の多くをシェルター内で過ごす。**必ず設置すること。**

- 個体の体がすべて入るサイズを選ぶ
- **ウェットシェルター**（上部に水を入れて内部の湿度を高めるタイプ）がおすすめ。脱皮のサポートにもなる
- シェルターの上部に常時水を入れておけば、別途水入れを設置しなくても問題ない

## パネルヒーター

夜間の保温に**パネルヒーターが必須**。エアコンのみでの温度管理は補助手段として有効だが、ケージ内の温度勾配を作るために専用のパネルヒーターをあわせて使用することを推奨する。

- ケージ床面の**1/3程度**の面積に敷く（全面に敷くと逃げ場がなくなる）
- シェルターの下半分にかかる位置に設置するのが理想
- 温湿度計でケージ内温度を定期的に確認する

## 照明

日光浴をしない種のため紫外線灯・バスキングランプは不要。観賞用の蛍光灯があれば十分。

## 温度・湿度

| 項目 | 目安 |
|------|------|
| ケージ内温度 | 26〜28℃（最低26℃以上を維持） |
| 湿度（通常時） | 60%前後 |
| 湿度（脱皮前） | 80%前後 |

- 冬眠させない場合は年間を通じて26℃以上を保つ
- 毎日霧吹きを行い乾燥を防ぐ（冬場は1日3〜4回、夏場は1日2回が目安）
- 湿気を保つ分汚れやすくダニが発生しやすいため、床材・シェルターの清潔を保つ

## 水入れ

- 全身が浸かれるサイズを用意する（脱皮前に全身浸かることがある）
- 倒れにくい重みのある容器を選ぶ
- 水は定期的に交換して清潔を保つ
- ウェットシェルターを使用している場合は上部の水が水入れを兼ねる
