# クレステッドゲッコー - モルフ一覧

## 概要
飼育の歴史こそ短いものの、赤や黄色など様々な体色をした個体を掛け合わせたモルフが産出されている。体色には変異があり、多彩な品種が流通している。モルフによって外見の印象が大きく変わり、人気や希少性によって価格帯も異なる。

## 代表的なモルフ

| モルフ名 | 特徴 | 価格帯の目安 |
|---|---|---|
| フレイム（またはファイア） | 背中が明るく縁取られ、炎のように見える | 20,000円〜 |
| ハーレークイン | 胴体や手足に複雑な模様が入り、個体ごとに変化が大きい | 40,000円〜 |
| ダルメシアン | 体に斑点が入るのが特徴。斑点の数で価値が変わる | 20,000円〜 |
| ピンストライプ | 背中に2本の線が通っている | 15,000円〜 |
| リリーホワイト | 白色の面積が多く、希少性が高い人気モルフ。白色の面積により価値が変わる | 40,000円〜 |
| オレンジ | 鮮やかなオレンジ色の体色 | — |
| タイガー | 縞模様が入る | — |
| レッド | 赤色の体色 | — |

※価格はあくまで目安であり、模様の入り方や血統によって大きく変動する。実際に購入する際は信頼できるブリーダーやショップを選ぶこと。

## 遺伝情報
クレステッドゲッコーのモルフは体色・模様の組み合わせによるもので、ヒョウモントカゲモドキほど単純メンデル遺伝で整理されていない。品種改良の歴史が進むにつれ、より多様な個体が生み出されている。
