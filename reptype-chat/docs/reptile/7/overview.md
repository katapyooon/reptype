# クレステッドゲッコー（type_id: 7）- 概要

## 基本情報

| 項目 | 内容 |
|---|---|
| 学名 | Correlophus ciliatus |
| 和名 | オウカンミカドヤモリ（王冠帝守宮） |
| 分類 | 有鱗目イシヤモリ科オウカンミカドヤモリ属 |
| 主な生息地 | ニューカレドニア（本島南部、パイン島および周辺の島々） |
| 活動時間 | 夜行性（日光浴も行う） |
| 体長・体重（成体） | 体長：約17〜20センチメートル、体重：約45〜50グラム |
| 寿命 | 7〜10年。良好な飼育環境下では15〜20年生きる場合もある |
| 性格 | おとなしく温厚で、人に慣れやすい。ただし、個体により程度が異なる |

## 歴史
1866年に記載されて以降、発見例が少なく、1994年に再発見されるまでは絶滅したと考えられていた。

## 名称の由来
種小名 *ciliatus* は「（多くの）繊毛を持つ」を意味する生物学用新ラテン語で、眼上部から背面にかけて並ぶ棘状の鱗を睫毛に見立てた名付けである。英語名 "crested gecko" はこの突起状の鱗をクレスト（紋章学のクレスト、転じて王冠の意）に見立てたもので、和名の「オウカン（王冠）」もここに由来する。日本では「クレステッドゲッコー」または略して「クレス」の愛称で親しまれている。

## 特徴と魅力
まつ毛のように見えるクレスト（突起）が印象的で、目元を柔らかく見せる。大きく猫のような目は愛嬌があり、舌をペロリと出す仕草もユニークである。体色や模様のバリエーションが豊富で、自分好みの個体を選べる点も人気の理由となっている。人に慣れやすい性格をしており、ハンドリング（手乗せ）を楽しめる点も魅力。樹上棲ヤモリの中でも人気が高く、飼育入門種とされることもある。

## 形態
全長はおよそ17〜20センチメートル。体色には変異がある。眼上部から背面にかけて棘状の鱗（クレスト）が並ぶ。尾にも趾下薄板と同じような吸着機能がある。尾を自切した場合、基部には玉状の組織が形成され、元のような長い尾には再生しない。

## 体のつくり

### 壁やガラスを登れる理由
指先に「趾下薄板（しかはくばん）」と呼ばれる特殊な構造をもち、細かい毛のような繊維がびっしりと並んでいる。分子同士の弱い引力（ファンデルワールス力）を利用して面に吸着するため、垂直な壁やガラス面を簡単に登ることができる。吸盤とは異なる仕組みであり、乾燥や汚れによってこの機能が弱まることもある。

### 尾の特徴
「フロッピーテール」と呼ばれる症状に注意が必要。これは尾が不自然に曲がった状態で定着してしまうもので、特にケージが狭く自由に動き回れない環境で発生しやすい。また、危険を感じると尾を自ら切り離す「自切（じせつ）」を行うが、一度切れた尾は再生しない。尾はバランスを保つ役割を果たしているため、できる限り自切を避ける環境づくりが大切である。

## 分布
ニューカレドニアの固有種で、ニューカレドニア本島南部、パイン島および周辺の島々に生息する。

## 分類

### 系統分類
かつては Rhacodactylus 属（ミカドヤモリ属）に分類されていたが、再分類の結果、オウカンミカドヤモリとベレプミカドヤモリに2分割され、さらにサラシノミカドヤモリを加えたうえで新属 *Correlophus* に納められた。

## ペットとしての流通
ペット用として飼育されることがあり、日本にも輸入されている。ニューカレドニアに生息する野生動物の輸出は禁止されているため、研究用に輸出された個体から繁殖した個体のみが流通する。かつては高価だったが、繁殖個体の流通増加により価格が落ち着き、専門店では店頭で普通に見かける種となっている。
