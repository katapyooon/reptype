# マレーキャットゲッコー（type_id: 12）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Gekko gecko* |
| 和名 | マレーキャットゲッコー |
| 英名 | Malay Cat Gecko |
| 分類 | 爬虫綱 有鱗目 ヤモリ科 |

## 分布

東南アジア（マレー半島・周辺地域）

## 形態

- 全長：最大30cm程度
- 体色：淡いグリーンからベージュに近い色合いが一般的
- 体表に特徴的な斑点や模様が現れることもある
- しっかりとした指先と強力な足のひらを持ち、垂直な壁を登ることができる優れた登攀能力を持つ

## 生態

- 東南アジアの熱帯地域に生息する
- 夜行性が強く、夜間に活発に活動する
- 昼間はじっとしていることが多い

## 性格・行動

- 非常におおらかな性格
- 警戒心が少なく、飼い主に慣れると手に乗せても安心して過ごす
- 昼夜のサイクルに適応している

## ボルネオキャットゲッコーとの違い

マレーキャットゲッコーと混同されがちな「ボルネオキャットゲッコー」も人気のあるペットだが、いくつかの明確な違いがある。

### サイズ
ボルネオキャットゲッコーは成体でも体長が25cm程度とやや小柄。マレーキャットゲッコーは最大30cmほどまで成長する。

### 色合い・模様
ボルネオキャットゲッコーはやや暗い色合いの個体が多く、灰色・茶色・オレンジ色の模様が見られる。マレーキャットゲッコーは淡い緑色に鮮やかな斑点が入ることが多い。

### 性格
マレーキャットゲッコーは比較的社交的でおおらかな性格である一方、ボルネオキャットゲッコーは少し警戒心が強く、慣れるまで時間がかかることがある。初めて爬虫類を飼う方にはマレーキャットゲッコーが飼いやすい選択肢となる。
