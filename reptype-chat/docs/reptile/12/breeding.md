# マレーキャットゲッコー - 繁殖

## 繁殖様式

卵生。

## 繁殖に適した環境

| 項目 | 目安 |
|------|------|
| 温度 | 28〜30℃ |
| 湿度 | 60〜70％ |
| 照明サイクル | 12〜14時間の明暗周期 |

## ペアリング

- オスとメスのペアが必要
- オスが求愛行動を示し、メスが反応することでペアリングが成立する
- 繁殖期には餌を多めに与え、ストレスを最小限に抑えることが重要
- 繁殖を強要せず、オス・メスともに健康であることを確認したうえでペアリングを行う
- 繁殖期にはペアにストレスを与えないよう、静かな場所に置くことが望ましい

## 産卵

- ペアリング後、メスは約2〜3週間の間に産卵する
- 通常2〜3個が1つの卵塊として産まれる

## 卵の管理・孵化

- 卵は湿度が高く温かい場所に保管し、乾燥しないよう管理する
- 専用の孵化用ケースを準備することが推奨される（エアコン・ヒーターの直接影響を避ける）
- 孵化までの目安：約60〜90日（温度・湿度の管理によって前後する）
- 定期的に卵の状態をチェックし、乾燥していないか確認する

## 幼体の育て方

- 孵化した幼体は非常にデリケートなので、温度・湿度が適切に維持された環境で管理する
- 最初の1〜2週間は細かい昆虫（粉末状のミールワームや小さなコオロギ）を与える
- 育成初期は湿度を高めに保ち、脱皮しやすい環境を整える
- 成長に伴い食欲も増すため、成体と同様にカルシウムやビタミンを補う
- 体調に問題がある場合は早めに獣医師に相談する
