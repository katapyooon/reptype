# マレーキャットゲッコー - 飼育方法

## 購入前に知っておくべきこと

### 価格・販売店
- 一般的な価格帯：約5,000円〜1万円程度
- 血統や模様、販売店によって価格が異なる
- 稀少個体や特別な模様を持つ個体は高価になることもある

### 購入場所
爬虫類専門店またはオンラインショップが一般的。信頼性のある販売店を選ぶために、レビューや評価を確認したり、飼育に関する質問を店員にしてみることを推奨する。

## ケージ

- 最低でも高さ40cm・幅60cm程度のものが理想
- マレーキャットゲッコーは垂直に登る能力を持つため、壁面を登れるように広めの格子があるものや足場となるアイテムを設置する

## 温度・湿度

| 項目 | 目安 |
|------|------|
| 日中温度 | 24〜28℃ |
| 夜間温度 | 22〜24℃ |
| 湿度 | 50〜70％ |

- 温度を安定させるため、ケージの一部にヒーターを設置し温度計で定期的に確認する
- ケージ内に温度のバリエーションを持たせ、ゲッコーが自分で温かい場所と涼しい場所を選べるようにする
- 湿度が低すぎると脱皮不良・乾燥による皮膚や呼吸器への負担が生じる
- 湿度が高すぎる場合はケージの蓋や側面を少し開け、空気の流れを作る

## 床材

ココナッツファイバー・樹皮・乾燥した葉っぱなどが推奨される。通気性が良く湿度を保持しやすいものを選ぶ。

## 照明

強い紫外線灯やバスキングランプは不要。自然光に近い明暗周期（12〜14時間）を維持する。

## 容器内レイアウト

- **足場・登れる場所**：木の枝・岩・網状の設置物を用意し、垂直移動ができる環境を整える
- **隠れ場所**：洞窟型のシェルターやコルクなどを設置する。温かい場所・涼しい場所の両方に用意することが望ましい
- **植物**：フェイクプランツでも可。ライブプランツを使うとより自然な環境を再現できる
- **水入れ**：清潔な水入れをケージ内に設置する

## 基本的な世話

定期的な霧吹きで湿度を補充する。ケージ内を定期的に清掃し、衛生的な環境を維持する。
