# ミヤコヒキガエル - 飼育方法

生息地である宮古島諸島は年間平均気温23℃・湿度80%の亜熱帯海洋性気候。最も寒い時期でも10〜15℃程度、夏は9月頃まで30℃以上が続く。この自生地の環境を踏まえながら飼育環境を整えることが基本となる。

## ケージ

- 地表性のカエルのため、床面積を重視した横長のケージが適している
- 複数飼育する場合はゆとりのある広さを確保する
- 狭すぎ・低すぎはNG。カエルはそれなりの跳躍力があり、ぶつけて怪我をする可能性がある

## 温度・湿度

| 項目 | 目安 |
|------|------|
| 温度 | 25〜30℃ |
| 湿度 | 60〜70% |

- **蒸れに弱い**ため、やや乾燥気味に管理する。湿度80%以上が長く続く場合はサーキュレーターや除湿機で対策する
- 乾燥しすぎもNGなため、温湿度計を設置して環境をこまめに確認する
- 急激な温度変化はストレスになるため、季節の変わり目は特に注意する

## 保温器具

冬に冬眠させないのであれば保温器具があると安心。

- **パネルヒーター**：ケージ底面や側面に設置して保温する
- **サーモスタット**：温度を自動調整し過加熱を防ぐ。あわせて使用を推奨

## 床材

以下のいずれかが使用されることが多い。

| 床材 | 特徴 |
|------|------|
| 黒土（園芸用） | 誤飲リスクが低く安価。定期的な全交換に向く |
| 赤玉土 | 通気性・保湿性のバランスが良い |
| フロッグソイル | 両生類向け専用。扱いやすい |
| ペットシーツ | 清潔管理しやすいが自然な感触は出にくい |

床材は2週間に1回程度を目安に全交換し、衛生を保つ。

## シェルター

隠れ家を設置して落ち着ける場所を作る。カエルが湿度を調整するためにも有効。

- 素焼き鉢・コルク・専用シェルターなど素材は問わない
- カエルが全身を隠せるサイズを選ぶ

## 水・水容器

- 全身が浸かれる浅い水容器を常設する（深すぎると溺れる危険があるため、顔が余裕で出せる深さにする）
- カエルはお腹の皮膚から水分を吸収するため水入れは必須
- 水道水を使用する際は必ず残留塩素を除去してから使用する（カルキ抜き）
- 水は2日に1回以上交換して新鮮な状態を保つ
- 飼育水のpHは7.2〜7.4を目安に管理する

## ハンドリング

- 触られることがストレスになるため、極力触らないようにする
- お迎え直後は特に強いストレスを感じているため、2〜3日は餌やりやハンドリングを控えてそっと見守る
- 皮膚から粘液や毒性物質を分泌するため、触った後は必ず手を洗う
- 粘膜（目・口・傷口）には触れないよう注意する

## 毒について

ヒキガエルは目の後ろの耳腺に毒（ブフォトキシン）を持っている。

- 通常の飼育では分泌されることはほとんどない
- 耳腺から出た白い分泌液に触れた場合は、目や口に触れないようにしてすみやかに手を洗う
- 体表のイボからも弱い毒性の粘液が出るため、触った後は必ず手洗いをすること

## 飼育上の注意

- **条例**：宮古島市での捕獲は条例により禁止されている
- **逸出防止**：国内で従来生息していなかった地域への人為的な移入が確認されているため、逃がさないよう特に注意する
- **混泳禁止**：近縁種のカエルと交雑することが確認されているため、他種との混泳は避ける
- **過給餌に注意**：食欲旺盛な個体はどんどん食べるため、体格に合わせた量を与える
- **拒食時**：ストレスや環境変化が原因のことが多い。嗜好性の高い餌を試す、おき餌にしてそっとしておくなど環境を整えることが優先
