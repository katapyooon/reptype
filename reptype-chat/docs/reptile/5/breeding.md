# ミヤコヒキガエル - 繁殖

## 繁殖期

繁殖期は長く，9月から3月にわたる。ニホンヒキガエルと異なり，1繁殖場所内での繁殖期も長い。

## 繁殖行動

- 止水に多数の雄が集まり，「クックックッ…」と聞こえる声で鳴く
- 繁殖場所はため池，防火用水等の止水

## 産卵

- ゼリー層に包まれた紐状の卵塊（卵紐）を産出する
- 蔵卵数：12,000〜14,000個

## 幼生・変態

- 幼生の多くは3月以降に変態する

## 飼育下での繁殖における注意

- 国内で従来生息していなかった地域への人為的な移入が確認されているため，繁殖させた個体を野外へ逃がさないよう特に注意が必要
- 近縁種のカエルと交雑することが確認されているため，他種との混泳・繁殖は避ける
