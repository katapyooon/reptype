# コーンスネーク - モルフ一覧

コーンスネークは品種改良が盛んで、多彩なカラー・パターンのモルフが存在する。

## カラーバリエーション

| モルフ名 | 特徴 | 遺伝形式 |
|---------|------|---------|
| ノーマル（レッド） | 赤〜オレンジの地色に黒い縁取り斑紋。最もスタンダード | - |
| アルビノ（アメラニスティック） | 黒色色素欠損。ノーマルから黒を抜いた状態。目が赤くなる | 劣性遺伝 |
| アネリスリスティックA（アネリ） | 赤色色素欠損。グレー〜ブラック系の体色。成長すると黄色を発色 | 劣性遺伝 |
| チャコール（アネリスリスティックB） | アネリBとも呼ぶ。成長しても黄色をほぼ発色しない | 劣性遺伝 |
| スノー | アルビノ × アネリA。黒と赤の色素がなくなり体が白くなる。パターンはうっすら残る | - |
| ブリザード | アルビノ × チャコール。スノーより白が強く、柄が薄くなる | - |
| ハイポメラニスティックA | 黒色色素減衰。赤や橙色が明色になり、黒色部が紫色や褐色になる | 劣性遺伝 |
| サンキスド（ハイポメラニスティックB） | オケーティーのハイポメラニスティック | 劣性遺伝 |
| オケーティー | 橙がかった赤の地色に濃赤色の斑紋。サウスカロライナ州Jasper郡の地域変異 | - |
| マイアミフェイズ | 灰褐色の地色に縁取りが不鮮明な赤い斑紋。フロリダ州南部（マイアミ周辺）の地域変異 | - |
| リバースオケーティー | アメラニスティックのブリーディングから産まれた品種。オケーティーの黒色部が白い | 劣性遺伝 |
| ブラッドレッド | 成長に伴い体色と斑紋の境界が曖昧になり、アダルトは赤一色になる | - |

## パターンバリエーション

| モルフ名 | 特徴 |
|---------|------|
| ブロッチ | 最も一般的なパターン。黒く縁取られた短形（ブロッチ）が連続する |
| ストライプ | ブロッチが直線状に変化。さらに薄れてほぼ柄のない「バニッシングストライプ」も存在 |
| モトレー | ブロッチが円形状に変化。腹部の市松模様もプレーンベリー（無地）になる |

## 主な組み合わせモルフ

- **ファイア**：ブラッドレッド × アルビノ
- **ピューター**：ブラッドレッド × チャコール（アネリB）

## 遺伝情報

コーンスネークのモルフは劣性遺伝が多く、ヘテロ（1コピー保有）では表現型に現れないが、次世代への遺伝子伝達に影響する。組み合わせ交配で複数のモルフを同時に発現させることができる。
