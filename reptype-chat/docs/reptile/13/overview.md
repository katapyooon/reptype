# コーンスネーク（type_id: 13）- 概要

## 基本情報

| 項目 | 内容 |
|------|------|
| 学名 | *Pantherophis guttatus* |
| 和名 | コーンスネーク、アカダイショウ |
| 英名 | Corn Snake |
| 分類 | 爬虫綱 有鱗目 ナミヘビ科 アメリカネズミヘビ属 |
| 原産地 | アメリカ合衆国南東部 |
| 全長 | 90〜120cm（最大183cm） |
| 平均寿命 | 10〜15年 |

アメリカネズミヘビ属の模式種。別名アカダイショウ。

## 分布

アメリカ合衆国南東部（ウエストバージニア州東部・ニュージャージー州南部・ペンシルベニア州南部以南、およびテネシー州・ミシシッピ州・ルイジアナ州以東）。

模式標本の産地はチャールストン（サウスカロライナ州）。

## 形態

- 全長90〜120cm、最大全長183cm
- 体列鱗数：25〜29
- 腹板数：203〜245
- 尾下板数：47〜84

体色は赤・橙色・黄色・赤褐色・灰色などと変異が大きく、体色よりも濃色の赤や橙色の斑紋が入る。背面の斑紋は黒く縁取られる個体が多い。腹面の体色は白く、黒い角張った斑紋が市松模様状に入る。

上唇板・下唇板は明色で暗色に縁取られる。左右の眼の間には三日月状の斑紋が入る。虹彩は赤や橙色で、瞳孔は丸い。

和名・英名の「コーン（corn）」は、腹部の市松模様がトウモロコシの粒に見えることが由来とする説がある。

## 分類

2002年に発表されたミトコンドリアDNAの分子系統解析により、新大陸に分布するナメラ属の一部を本種を模式種としたアメリカネズミヘビ属 *Pantherophis* として分類する説が提唱された。

同年の別の分子系統解析から、以下の近縁種が独立種として整理された。

- プレーンズコーンスネーク *E. emoryi*（亜種から独立種へ）
- ルイジアナコーンスネーク *E. slowinskii*（新種記載）

## 生態

- 主に松林などの森林に生息するが、様々な環境に適応する
- 地表棲だが樹上に登ることもある
- 主に薄明時や夜間に活動する
- 地面の穴、石・倒木の下などで休む
- 分布域北部の個体群は冬季に冬眠する

**食性：** 小型哺乳類（特に齧歯類）、鳥類やその卵、爬虫類、両生類など。成蛇は主に小型哺乳類を捕食する。

**繁殖：** 卵生。1回に12〜25個（最大約30個）を産卵。卵は60〜75日で孵化する。

## 人間との関係

温和な性格と寒さへの強さからペットとして人気が高く、日本にも輸入されている。品種改良が盛んで多彩なモルフが存在し、ビギナーからエキスパートまで幅広い層に親しまれている。

### 主な品種（モルフ）

- **アメラニスティック（アルビノ）** - 黒色色素欠損。劣性遺伝
- **リバースオケーティー** - アメラニスティック由来の品種
- **アネリスリスティック** - 赤色色素欠損により赤みや橙色みがなくなる。劣性遺伝
  - アネリスリスティックA：成長に伴い黄色を発色する
  - チャコール（アネリスリスティックB）：成長しても黄色をほぼ発色しない
- **オケーティー** - 体色が橙がかった赤で斑紋が濃赤色。サウスカロライナ州Jasper郡の地域変異に由来
- **ハイポメラニスティック** - 黒色色素減衰。赤や橙色部分が明色になり、黒色部が紫色や褐色になる
  - ハイポメラニスティックA：一般的なハイポメラニスティック
  - サンキスド（ハイポメラニスティックB）：オケーティーのハイポメラニスティック
- **マイアミフェイズ** - 灰褐色の体色に赤く縁取りが不鮮明な斑紋。フロリダ州南部の地域変異に由来
