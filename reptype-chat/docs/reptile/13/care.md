# コーンスネーク - 飼育方法

## ケージ

アダルトの全長は120〜150cm程度だが、とぐろを巻くためそれほど広いスペースは不要。

- **ベビー〜ヤング**：幅40〜60cm程度のプラケース・アクリルケースで問題ない
- **アダルト**：最終的には幅90cm程度の大型ケース・爬虫類専用ケージが理想

脱走が得意な種のため、蓋がしっかりロックできるタイプを選ぶこと。

## 温度管理

変温動物のため、ケージ内に温かいエリアと涼しいエリアの温度勾配を作り、ヘビが自分で快適な場所を選べるようにする。

| エリア | 温度目安 |
|--------|---------|
| 温かいエリア（ホットスポット） | 28〜30℃ |
| 涼しいエリア（クールスポット） | 22〜25℃ |

### 保温器具

- **パネルヒーター／フィルムヒーター**：ケージ底面の1/2〜3/4に敷いて底面から温める（全面に敷くとヘビが逃げ場を失う）
- **保温球／セラミックヒーター**：上部から暖めるタイプ。冬場など底面ヒーターだけでは温度が不足する場合に併用する
- **サーモスタット**：温度を自動調整し過加熱を防ぐ。できれば使用を推奨

温度が低すぎるとエサ食いが悪くなり消化不良を起こしやすい。寒さには強く、冬季休眠（ウィンタークーリング）させることも可能だが、ベビーは避け1年以上育てたイヤリングサイズ以上から行うのが望ましい。

## 湿度管理

- 適正範囲：40〜60%
- 脱皮前後は湿度を一時的に上げる（脱皮不全の防止）
- 湿度計を設置し、霧吹きやウェットシェルターで調整する

## シェルター

臆病な性格のため、身を隠せる場所を必ず用意する。ケージ内に1つ以上設置すること。

- **ドライシェルター**：通常の隠れ家。ホットスポット側・クールスポット側の両方に置くとよい
- **ウェットシェルター**：内部に湿らせたミズゴケなどを入れたもの。脱皮のサポートになるためとくに推奨

シェルターはヘビが全身をすっぽり入れられるサイズを選ぶ。陶器製・樹脂製・コルクなど素材は問わない。

## 床材

| 床材 | 特徴 |
|------|------|
| 新聞紙・キッチンペーパー | 安価で交換しやすい。汚れが見えやすく清潔を保ちやすい。初心者向け |
| ウッドシェイブ（アスペンチップ） | 吸湿性があり自然な見た目。汚れた箇所だけ部分交換可能。ヘビが潜れる |
| ヤシガラマット | 保湿性が高く脱皮期に向く。ダニ対策が必要 |
| ペットシーツ | 衛生管理しやすい |

細かい粒状の床材は誤飲リスクがあるため、給餌時は床材の上に直接餌を置かず別容器で行うのが望ましい。

## 水入れ

- 全身が浸かれる大きさのものを選ぶ。脱皮前や暑い時期に水浴びすることがある
- 安定感があり倒れにくい重めのものを選ぶ
- 水は少しぬるめが適温（冷水は体にショックを与えることがある）
- 毎日交換して清潔を保つ

## 温度計

- ヒーターの真上は誤計測になるため、温度が安定している場所に設置する
- センサーが吸盤取り付け式の場合、パネルヒーターより少し上でヘビが活動する高さに設置する

## ピンセット

- 給餌専用。ある程度の長さがあるとアダルトになっても使いやすい
- 竹製・ステンレス製どちらも可

## ハンドリング

コーンスネークは温和な性格で、人間に攻撃的になることはほとんどない。

- 臆病な生き物なので、優しく扱えば噛まれることはまずない
- ある程度は飼い主に慣れるようになる
- **給餌直後は触らない**（腹部を刺激すると吐いてしまうことがある）

## セッティング例

1. パネルヒーターをケースの1/2〜3/4の面積の下に敷く
2. 床材（ウッドシェイブ、新聞紙など）をセット
3. シェルターを設置（ドライ・ウェット各1つが理想）
4. 全身が浸かれる水入れを設置
5. 温度計を安定した位置に設置
6. ケージ内の温度勾配（28〜30℃ ／ 22〜25℃）を確認する
